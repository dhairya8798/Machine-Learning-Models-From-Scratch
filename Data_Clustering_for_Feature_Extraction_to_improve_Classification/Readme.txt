The zip file contains 3 files:
-	Project_dxp8341.ipynb (Python code)
-	log.csv (dataset required for the python code)
-	Readme.txt (Instructions to run the code)

To run the code:

-	Open the ipynb file in Google Colab or Jupyter Notebook

-	For Google Colab:
	-	Upload the csv file on Files section of sessions storage
	-	Select "Run all" in Runtime

-	For Jupyter Notebook:
	-	The csv and ipynb file should be in the same folder
	-	Select "Run all" in Runtime
